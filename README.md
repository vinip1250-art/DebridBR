# Brazuca + Debrid Addon

Addon para Stremio que envolve resultados do Brazuca Torrents com Real-Debrid, Torbox e StremThru.

## Instalação

1. Clone este repositório.
2. Configure variáveis de ambiente no Vercel:
   - `USE_RD=true`
   - `USE_TORBOX=true`
   - `USE_STREMTHRU=true`
   - `RD_API_TOKEN=...`
   - `TORBOX_TOKEN=...`
3. Deploy no Vercel.
